### thunk

thnuk提供了redux异步更新的功能，在配置thunk中间件之后，reducer中传入的action，会经过中间件的处理，所以可以传入函数或者promise等作为action参数

```jsx
const thunkMiddleware =
  ({ dispatch, getState }) =>
  next =>
  action => {
    if (typeof action === 'function') {
      return action(dispatch, getState)
    }

    return next(action)
  }
```
